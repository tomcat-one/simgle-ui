## 森果伟创 vue3 组件库

### 安装

```
npm install simgle-ui
```

### 使用

#### 1.全量使用 main.js 中导入

```
import simgleUI from 'simgle-ui'
import 'simgle-ui/es/style.css'

app.use(simgleUi)
```

#### 2.按需加载 用什么导入什么，样式需要先引入到 main.js

```
import 'simgle-ui/es/style.css'
```

```
import { uvButton } from 'uv-ui'
```
